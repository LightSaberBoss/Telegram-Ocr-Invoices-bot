## 📋 Описание проекта

Construction Receipt Parser - инструмент для автоматического распознавания и обработки чеков из строительных магазинов. Использует AI для извлечения данных и категоризации расходов.

### 🎯 Основные возможности

- OCR распознавание чеков
- Извлечение товаров и цен
- Автоматическая категоризация
- Экспорт в Excel/CSV
- Статистика расходов

### 🛠️ Технологии

- **OCR**: Tesseract, Google Vision API
- **AI**: OpenAI для категоризации
- **Backend**: Python, FastAPI
- **Frontend**: React, TypeScript
- **База данных**: SQLite/PostgreSQL

### 📁 Структура проекта
